# 美术需求文档 v1.0

---

## 1. 概述

本文档定义了**轮椅大作战**游戏的美术资源规范，包括角色设计、UI设计、图标规范、动画要求等。所有美术资源需遵循本文档规范进行制作。

---

## 2. 视觉风格

### 2.1 整体风格定位
**"土味酷炫"** — 既接地气又有潮流感

- 色彩饱和度高，对比强烈
- Q版2D角色，圆润可爱
- 表情夸张，动作魔性
- 服装配色大胆（土红配翠绿、花衬衫等）
- UI圆润设计，少量毛笔字元素

### 2.2 参考风格
- 《搞怪大作战》的搞笑风格
- 抖音土味视频的视觉冲击
- 广场舞服装的鲜艳配色

### 2.3 禁止风格
- 血腥暴力内容
- 政治敏感元素
- 低俗色情内容

## 4. UI设计规范

### 4.1 界面尺寸

| 界面 | 尺寸 | 说明 |
|------|------|------|
| 主界面 | 750x1334 | iPhone 6/7/8尺寸 |
| 游戏画布 | 750x1334 | 全屏 |
| 图标 | 48x48, 64x64, 96x96 | 按钮图标 |
| 头像 | 80x80, 120x120 | 用户头像 |

### 4.2 颜色规范

#### 主色调
```
主题色:     #FFD700 (金色)  - 用于高亮、按钮、进度条
辅助色:     #FF6666 (红色)  - 用于警告、血量、攻击按钮
强调色:     #66CCFF (蓝色)  - 用于技能、特效
背景色:     #333366 (深蓝)  - 用于主背景
次背景色:   #444488 (蓝紫)  - 用于卡片、面板
文字主色:   #FFFFFF (白色)  - 用于主要文字
文字次色:   #CCCCCC (灰色)  - 用于次要文字
```

#### 血条颜色
```
满血:     #66FF66 (绿色)
中血:     #FFFF66 (黄色)  
低血:     #FF6666 (红色)
护盾:     #66CCFF (蓝色)
```

### 4.3 字体规范

| 用途 | 字体 | 大小 | 颜色 |
|------|------|------|------|
| 标题 | 站酷快乐体/PingFang Bold | 48-56pt | #FFD700 |
| 副标题 | 站酷快乐体/PingFang Bold | 36pt | #FFFFFF |
| 正文 | PingFang/PingFang SC | 28pt | #FFFFFF |
| 按钮 | PingFang Bold | 32pt | #FFFFFF |
| 数字 | DIN Alternate | 32-48pt | #FFD700 |

### 4.4 UI组件规范

#### 按钮
- 尺寸: 300x80px (主按钮), 200x60px (次按钮)
- 圆角: 40px
- 背景: 渐变 (#FF6666 → #FF9966)
- 阴影: 0 4px 15px rgba(255,102,102,0.3)
- 点击效果: 缩小95%, 透明度80%

#### 卡片
- 圆角: 20px
- 背景: rgba(255,255,255,0.1)
- 边框: 1px solid rgba(255,255,255,0.1)
- 间距: 20px

#### 进度条
- 高度: 16-20px
- 圆角: 8-10px
- 背景: rgba(255,255,255,0.2)
- 填充: 渐变色

- 最大尺寸: 200x200px

- 树木、花草、长椅、路灯等
- PNG格式，透明背景

## 8. 动画要求

### 8.1 角色动画

#### 动作循环 (6帧/秒)
- 待机: 轻微左右晃动
- 移动: 轮椅滚动 + 身体晃动
- 攻击: 手臂挥动 + 物品飞出

#### 一次性动画
- 受伤: 闪红 + 后退
- 死亡: 摔倒 + 淡出
- 胜利: 举起双手 + 欢呼

### 8.2 UI动画

| 动画 | 时长 | 缓动 |
|------|------|------|
| 按钮点击 | 0.1s | ease-out |
| 面板弹出 | 0.3s | bounce |
| 数字变化 | 0.2s | ease-out |
| 进度条 | 0.3s | linear |
| 页面切换 | 0.3s | ease-in-out |

### 8.3 粒子效果

| 效果 | 参数 |
|------|------|
| 鸡蛋爆炸 | 数量10, 速度5-15, 生命周期0.5s |
| 技能释放 | 数量30, 速度3-8, 生命周期1s |
| 行走尘土 | 数量3, 速度1-3, 生命周期0.3s |

## 10. 资源输出规范

### 10.1 文件命名

```
类型_名称_序号.png/jpg/mp3

示例:
char_li_idle_01.png     (角色待机帧)
char_li_attack_01.png   (角色攻击帧)
weapon_egg_icon.png     (武器图标)
bg_community_01.png     (背景图)
sfx_throw.mp3           (音效)
```

### 10.3 技术规格

| 类型 | 格式 | 限制 |
