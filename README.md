# 🥚 轮椅大作战 (Wheelchair Wars)

> 年轻人的轮椅Battle - 当不婚不育的年轻人老了以后...

## 🎮 游戏简介

**轮椅大作战** 是一款多人对战街机游戏，故事背景设定在不婚不育的年轻人老去后的未来。玩家将操控坐着轮椅的老大爷老大妈，穿着酷炫的衣服互相投掷鸡蛋、拐杖、假牙等"武器"，展开一场场欢乐又魔性的轮椅大战！

### 核心玩法
- 🎯 1v1 实时对战
- 🎮 虚拟摇杆控制移动，点击投掷攻击
- ⚡ 技能系统 - 大招充能释放
- 🏆 排位赛 - 青铜到王者段位

## ✨ 游戏特色

- **魔性搞笑的题材**：戳中当代年轻人痛点
- **简单易上手**：虚拟摇杆 + 点击投掷
- **6个独特角色**：每个角色有专属技能
- **8种武器**：鸡蛋、铁蛋、拐杖、假牙飞弹...
- **4张地图**：社区广场、菜市场、公园、医院
- **社交分享**：炫耀战绩，邀请好友对战

## 📁 项目结构

```
wheelchair_wars/
├── miniprogram/           # 微信小程序代码
├── game/                  # Cocos 游戏项目
├── docs/                  # 文档
├── resources/             # AI生成的资源
└── tools/                 # 构建工具
```

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 开发调试
npm run dev

# 构建生产版本
npm run build
```
